Slide Types and Sections
========================================================

You can specify a type for a slide which changes it's default appearance. There are four built-in types:

- section
- sub-section
- prompt
- alert


New Section
====================================
type: section


New Sub-Section
====================================
type: sub-section


Prompt Slide
====================================
type: prompt




Alert Slide
====================================
